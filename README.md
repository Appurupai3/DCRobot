# DCRobot
# Discord Valorant & Economy Bot

這是一個結合 **Valorant (特戰英豪) 戰績查詢** 與 **經濟博弈系統** (賽馬、骰子、駭客遊戲等) 的 Discord 機器人。

## 📋 前置需求

- **Python 3.8** 或以上版本
- 一個 Discord 機器人帳號 (Token)
- Riot Games API Key (用於查詢戰績)

## 🛠️ 安裝步驟

1. **下載程式碼**
   將專案下載到你的電腦或伺服器。

2. **安裝依賴套件**
   開啟終端機 (Terminal / CMD)，進入專案資料夾，執行以下指令：
   ```bash
   pip install -r requirements.txt
   python bot.py
